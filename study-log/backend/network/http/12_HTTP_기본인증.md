# HTTP 기본인증 프레임워크 
- 인증 요청
  - WWW-Authenticate + 401
    - 보안 영역(realm)별 다른 사용자 권한 필요 
- 이름과 비밀번호 암호화해서 전달
  - Authorization 헤더
  - base64(이름:비밀번호): 8bit 시퀀스를 6비트 시퀀스로 변환, 어떤 문자든 복원가능
- 프록시에서 접근정책 중앙관리 가능

# 기본 인증 보안 결함
- 비밀번호 쉽게 디코딩
- 비밀번호 가로채서 위장
- 헤더 값 건드려서 트랜잭션 변형
- 가짜 서버 위장
